# Contributing to Archon

Thank you for your interest in contributing to Archon! We welcome contributions from everyone. By participating in this project, you agree to abide by our Code of Conduct.

### Reporting Bugs
If you find a bug, please create a GitHub issue including:
- Steps to reproduce the bug
- Expected vs. actual behavior
- Screenshots or logs if applicable

### Suggesting Enhancements
We love new ideas! Please open an issue to discuss your feature request before working on it.

### Pull Requests
1.  **Fork the repository** and create your branch from `main`.
2.  **Clone the repository** to your local machine.
3.  **Create a new branch** for your feature or bugfix:
    ```bash
    git checkout -b feature/amazing-feature
    ```
4.  **Commit your changes** with clear, descriptive messages.
5.  **Push to your fork** and submit a Pull Request.
